# <a name="manage-surface-pro-x-devices"></a>Zarządzanie urządzeniami Surface Pro X

Jeśli dołączysz urządzenia Surface X Pro do usługi Azure Active Directory (Azure AD), możesz zarządzać urządzeniami połączonymi za pomocą usługi Microsoft Intune. Usługa Intune umożliwia administratorom zdalne wdrażanie aplikacji i ustawianie wskazówek w celu monitorowania sposobu, w jaki dane organizacyjne są udostępniane i sprawdzane w organizacji.

Można zarządzać urządzeniami Surface Pro X za pomocą rozwiązania MDM (Mobile Device Management). Aby dowiedzieć się więcej na temat funkcji MDM i sposobu zarządzania urządzeniem Surface Pro X, zobacz Zarządzanie urządzeniem [Surface Hub za pomocą dostawcy usługi MDM.](https://docs.microsoft.com/surface-hub/manage-settings-with-mdm-for-surface-hub)

Niektórych programów antywirusowych innych firm nie można instalować na urządzeniach Surface X Pro. Oprogramowanie antywirusowe Windows Defender pomaga chronić Twoje urządzenia przez cały okres istnienia urządzenia. Aby dowiedzieć się więcej, skontaktuj się z dostawcą oprogramowania antywirusowego.
