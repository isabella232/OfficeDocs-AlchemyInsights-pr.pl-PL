# <a name="how-to-change-your-theme"></a>Jak zmienić motyw

Aby zmienić motyw w programie Outlook.com lub aplikacji Outlook w sieci Web, przejdź do obszaru **Ustawienia** i wybierz motyw w obszarze **motyw**. Aby wyświetlić wszystkie dostępne motywy, wybierz pozycję **Wyświetl wszystko** w obszarze **motyw**.
